# 📊 Análise de Vendas por Cidade com SQL

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em SQL voltadas para análise de dados.

## 🧠 O que foi analisado:

- Quantidade de clientes por cidade
- Total de vendas por região
- Média de gasto por cliente
- Identificação de clientes com maior consumo
- Total de pedidos por cidade (incluindo cidades sem pedidos)

## 🛠 Tecnologias utilizadas:

- MySQL Workbench
- SQL

## 📈 Principais conceitos aplicados:

- JOIN (INNER e LEFT)
- GROUP BY
- HAVING
- Subqueries
- Funções agregadas (COUNT, SUM, AVG)

## 🎯 Objetivo:

Simular análises reais de negócio que auxiliam na tomada de decisão, como identificar regiões com maior faturamento e clientes mais valiosos.

## 📚 Aprendizados:

- Utilização de JOIN para combinar dados de múltiplas tabelas
- Uso de GROUP BY para gerar análises agregadas
- Aplicação de HAVING para filtrar resultados agrupados
- Uso de subqueries para análises mais complexas
